Messaging App

Background

Freshman CS 180 - Object Oriented Programming. Final group project.

Creating a Seller

In the clip above, these things are happening:

  • User text file is empty, then the server and main class are ran
  • User of type Seller is created with stores "Fish" and "Potatoes" associated with the user
  • After server is closed, the user's information is stored in a text file

Creating a Customer to Buy From a Seller

In the clip above, these things are happening:

  • Conversation list text file is empty, then server and main class are ran
  • User of type Customer is created who messages the seller with store "Fish"
  • Seller account is signed into and replies back to customer
  • Customer account is signed into and replies back to seller
  • After server is closed, a new text file is created to save the conversation and the conversation is added to conversation_list text file
